Course Syllabus for Software System Security

Course Information

  • Course Code: 080660165
  • Course Title: Software System Security
  • Total Credits: 2
  • Lecture Hours: 18
  • Practical Hours: 36
  • Course Instructor: LIN Yan
  • Prerequisite: Operating Systems Principles

Course Overview

The course delves into software system security, focusing on common software vulnerabilities and their defenses. Students will learn how to identify, protect, and strengthen the security of software systems. Topics include deep analysis of software vulnerabilities, exploitation techniques, and defensive strategies, providing a strong foundation for future research or work in the field.

Course Objectives

  • Objective 1: Grasp the basic theories and engineering principles of software system security.
  • Objective 2: Address complex problems in fields such as cryptography, system security, and application security.
  • Objective 3: Combine theoretical knowledge with engineering practice to analyze complex issues in cybersecurity.

Practical Sessions

  1. Lab 1: Reverse engineering and debugging basics
  2. Lab 2: Using dynamic debugging tools
  3. Lab 3: Parsing binary file formats
  4. Lab 4: Stack and function call analysis
  5. Lab 5: Buffer overflow exploitation
  6. Lab 6: Code injection attacks
  7. Lab 7: Format string vulnerability exploitation
  8. Lab 8: Ret2libc attacks
  9. Lab 9: Return-oriented programming (ROP) attacks
  10. Lab 10: Software security defenses

Grading

  • Final Exam: 60%
  • Lab Sessions and Reports: 40%

Assessment Criteria

The evaluation includes understanding core concepts, practical skills, and final examinations. The course heavily emphasizes both theoretical understanding and hands-on application through labs.
